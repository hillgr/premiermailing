DATABASE LAYOUT SPECIFICATION


In order to provide you with the fastest service possible, please provide a database following the guidelines outlined below. Note: It is not a problem if your database includes additional non-mailing information in separate fields.


Name Information:

» First and last names: May be in the same or separate fields

» Additional optional fields: Salutation, Suffix, Title, Company Name


Address Information:


» Street address: May have two fields: delivery and alternate address field

» Street address: Should be in a separate field from city, state and zip code

» City, state & zip code: May be one or several fields

» Include country if foreign records included

File type accepted:

Excel, comma- or tab-delimited file

LAST NAME
